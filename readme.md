# DNA Life Organisation

A website built for a NGO called DNA Life Organisation under a 3-month internship as a full stack web developer.

Responsibilities given were to build a dynamic full stack mobile responsive website which allowed the organisation to share valuable information with their audience.

Technologies Used ( MERN Stack ) :-

1. MongoDB ( Atlas )
2. Express.js
3. React.js
4. Node.js
5. Material UI
6. SCSS
7. Firebase ( Authentication & Storage Buckets )
8. AWS ( Backend Hosting )
9. Cpanel ( Frontend Hosting )

Website Link : http://dnalife.org
